# Motor Vehicle Collision Analysis (NYC)

This project is a research study for **DAS7000 - Data Analytics & Visualisation**. It focuses on analyzing New York City's Motor Vehicle Collisions to provide data-driven insights for a fictional car insurance company, **MetroGuard Auto**.

## Project Objectives
- Analyze crash patterns, contributing factors, and injury/fatality trends across NYC's five boroughs.
- Develop a comprehensive dataset and visualization suite to assist in risk assessment for insurance purposes.
- Create an accompanying interactive **Tableau** dashboard for high-level stakeholder reporting.

## Dataset
- **Source:** [NYC Open Data - Motor Vehicle Collisions Crashes](https://catalog.data.gov/dataset/motor-vehicle-collisions-crashes)
- **Size:** ~2.2 Million Records / 29 Columns
- **Key Dimensions:**
  - **Temporal:** Crash Date and Time
  - **Geospatial:** Boroughs, Zip Codes, and Coordinates
  - **Severity:** Injuries and fatalities of pedestrians, cyclists, and motorists
  - **Categorical:** Contributing factors and vehicle types

## Tools & Technologies
- **Data Engineering:** Python (Google Colab, Pandas, NumPy)
- **Visual Analytics:** Altair, Vega Fusion, Vega-Lite
- **Dashboarding:** Tableau
